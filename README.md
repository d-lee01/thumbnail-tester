# HX Thumbnail Tester

A tool to test YouTube video thumbnail and title ideas before production.

## How It Works

1. **Admin uploads thumbnails** (admin.html)
2. **Team members test** (index.html) - select top 3, give feedback
3. **Results collected** in Google Sheets for analysis

## Quick Start

### 1. Set Up Google Sheets (One-time setup)

To collect responses in a Google Sheet:

1. **Create a Google Apps Script:**
   - Go to https://script.google.com
   - Click "New Project"
   - Paste this code:

```javascript
function doPost(e) {
  const sheet = SpreadsheetApp.openById('YOUR_SHEET_ID').getActiveSheet();

  try {
    const data = JSON.parse(e.postData.contents);

    // Add header row if empty
    if (sheet.getLastRow() === 0) {
      sheet.appendRow([
        'Timestamp', 'Name', 'Email', 'Age Group', 'Gender', 'Department',
        'Video Title', 'Rating'
      ]);
    }

    // Add one row for each rated video
    data.ratings.forEach(rating => {
      sheet.appendRow([
        data.timestamp,
        data.name,
        data.email,
        data.ageGroup,
        data.gender,
        data.department,
        rating.video,
        rating.rating
      ]);
    });

    return ContentService.createTextOutput(JSON.stringify({result: 'success'}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(error) {
    return ContentService.createTextOutput(JSON.stringify({result: 'error', error: error.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
```

2. **Get your Sheet ID:**
   - Create a new Google Sheet
   - Copy the ID from the URL: `https://docs.google.com/spreadsheets/d/[THIS_IS_YOUR_SHEET_ID]/edit`
   - Replace `YOUR_SHEET_ID` in the script above

3. **Deploy as Web App:**
   - Click "Deploy" > "New deployment"
   - Type: "Web app"
   - Execute as: "Me"
   - Who has access: "Anyone"
   - Click "Deploy"
   - Copy the Web App URL

4. **Add URL to index.html:**
   - Open `index.html`
   - Find `const SHEET_URL = 'YOUR_GOOGLE_APPS_SCRIPT_URL_HERE';`
   - Replace with your Web App URL

### 2. Upload Thumbnails

1. Open `admin.html` in your browser
2. Drag and drop thumbnail images
3. Add titles for each video
4. Click "Publish Test"
5. Save the downloaded `data.json` file to the project folder

### 3. Share with Team

1. Host the files (GitHub Pages, Netlify, etc.)
2. Share the tester URL with your team
3. They'll see randomized thumbnails and can vote

### 4. View Results

Open your Google Sheet to see:
- Who rated each video
- Their demographic info
- Individual ratings for each video (love, interesting, meh, wouldn't click, actively dislike)
- Easy-to-analyze data format

## Features

✅ Google Sign-In (captures email automatically)
✅ Demographic data collection (age, gender, department)
✅ Randomized order (removes position bias)
✅ YouTube-style grid layout
✅ 5-level rating system per video (Would definitely click → Actively dislike)
✅ Visual feedback with color-coded ratings
✅ Results exported to Google Sheets

## Tech Stack

- Pure HTML/CSS/JavaScript (no frameworks needed)
- Google Apps Script (for Sheets integration)
- GitHub Pages compatible

## Tips

- **Test with 10-15 thumbnails max** for best results
- **Use realistic thumbnails** that match your production quality
- **Randomization is automatic** - no need to worry about order bias
- **Track multiple rounds** by creating new Google Sheets for each test
- **Export results** from Google Sheets for deeper analysis

## Troubleshooting

**Thumbnails not loading:**
Make sure `data.json` is in the same folder as `index.html`.

**Google Sheets not receiving data:**
Check that you've updated the `SHEET_URL` in `index.html` and that your Apps Script is deployed as "Anyone can access".
